# YAML

## Types

Primitive:

- Integer
- Float
- String
- Boolean
- Date

Collection:

- Sequence
- Hash

Example:

```yaml
# Primitive
string:
  double-quote: "No one can know, not even Squidward's house"
  single-quote: 'Is mayonnaise an instrument?'
  no-quote: Finland
number:
  int: 24
  float: 3.5
bool: true
date: 2022-10-11
# Collection
sequence:
  sequence1: [1, 2, 3]
  sequence2:
    - 1
    - 2
    - 3
hash:
  a: 1
  b: 2
  c: 3
```
