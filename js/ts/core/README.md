# Typescript

Basic ts usage

## Setup

- `npm i`

## Startup (dev)

- `npm run dev`

## Startup (prod)

- `npm run build`
- `npm start`

### Usage

Install:

```sh
# install typescript with node type annotations
npm i -D typescript @types/node
```

Compile:

```sh
tsc helloworld.ts
```
