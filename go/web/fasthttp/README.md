# FastHTTP

https://github.com/valyala/fasthttp

## Install

go get -u github.com/valyala/fasthttp

## Startup

```bash
# dev (requires "air")
make dev
# run
make run
# build / run
make build; make run-build
```

Live Reloading Options:

- Air
  - https://github.com/cosmtrek/air
- Gin
  - https://github.com/codegangsta/gin
- Fresh
